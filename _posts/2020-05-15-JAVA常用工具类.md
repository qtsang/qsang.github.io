把日常的java工具类记录下来，方便以后查询，所谓好记性，不如烂笔头。

## 常用命令

1、记录方法运行时间

```java
Instant start = Instant.now();
// 业务代码
Instant end = Instant.now();
Duration duration = Duration.between(start, end);
LOGGER.debug("执行token为时间： " + duration.toMillis() + "ms");
```
